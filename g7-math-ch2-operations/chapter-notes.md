## 2.1 有理数的加法与减法

### 2.1.1 有理数的加法

#### 一句话目标
我能说出加法法则（同号/异号），会先定符号、再算绝对值。

#### 为什么需要加法法则

生活里到处是"相反意义的量"的相加：
- **收入 + 支出**：18.5（收入）+ (-6.5)（支出）= 12.0（结余）
- **温度变化**：零下 3℃ 上升 7℃ → (-3) + 7 = 4℃
- **运动方向**：先向右 5m，再向左 3m → 5 + (-3) = 2（在起点右侧 2m）

小学只学过"正数 + 正数"，现在要推广到**有符号的数相加**。

#### 直观模型：数轴上的运动

向右为正，向左为负。

**情况 1：同向运动（同号相加）**
- 向右 5m，再向右 3m：5 + 3 = 8（仍在向右）
- 向左 5m，再向左 3m：(-5) + (-3) = -8（仍在向左）

**结论**：**同号两数相加，和取相同符号，且和的绝对值等于加数绝对值的和。**

**情况 2：反向运动（异号相加）**
- 向左 3m，再向右 5m：(-3) + 5 = 2（最后在起点右侧 2m）
- 向右 3m，再向左 5m：3 + (-5) = -2（最后在起点左侧 2m）

**结论**：**异号两数相加，和取绝对值较大的加数的符号，且和的绝对值等于较大绝对值减较小绝对值。**

**特殊情况**：
- 互为相反数相加：5 + (-5) = 0（回到起点）
- 与 0 相加：5 + 0 = 5，(-5) + 0 = -5（位置不变）

#### 加法法则（三条）

1. **同号两数相加**：取相同符号，并把绝对值相加
   - 例：(-3) + (-9) = -(3+9) = -12

2. **异号两数相加**：取绝对值较大数的符号，用较大绝对值减较小绝对值
   - 例：12 + (-8) = +(12-8) = 4
   - 例：(-4.7) + 3.9 = -(4.7-3.9) = -0.8
   - 特例：互为相反数相加得 0

3. **一个数与 0 相加**：仍得这个数
   - 例：(-8) + 0 = -8

#### 方法：两步走

**"先定符号，再算绝对值"**：
- 第一步：看两个数是同号还是异号 → 确定结果的 + 或 -
- 第二步：根据法则计算绝对值

#### 例题

**例 1**：计算 (-3) + (-9)

解：① 同号 → 负号
    ② |(-3)| + |(-9)| = 3 + 9 = 12
    ③ (-3) + (-9) = -12

**例 2**：计算 12 + (-8)

解：① 异号，|12| > |-8| → 正号
    ② |12| - |-8| = 12 - 8 = 4
    ③ 12 + (-8) = +4 = 4

**例 3**：计算 (-4.7) + 3.9

解：① 异号，|-4.7| > |3.9| → 负号
    ② |-4.7| - |3.9| = 4.7 - 3.9 = 0.8
    ③ (-4.7) + 3.9 = -0.8

#### 易错提醒

1. **异号相加定错符号**：(-3) + 5 不是 -8，是 2（看绝对值大的那个数的符号）
2. **忘记特殊 0**：0 与任何数相加，都得这个数
3. **混淆绝对值运算**：(-3) + (-9) 不是 -12 而是 -(3+9) = -12

### 2.1.2 加法运算律

#### 一句话目标
我会用交换律、结合律简化计算。

#### 为什么运算律有用

小学学过的加法交换律、结合律，在有理数范围内**仍然成立**。

**加法交换律**：a + b = b + a（交换加数位置，和不变）

**加法结合律**：(a + b) + c = a + (b + c)（先加哪两个，和不变）

#### 应用：凑整、凑 0

**技巧 1：凑相反数得 0**
- 8 + (-6) + (-8) = [8 + (-8)] + (-6) = 0 + (-6) = -6

**技巧 2：同号结合、凑整**
- 16 + (-25) + 24 + (-35)
  = (16 + 24) + [(-25) + (-35)]
  = 40 + (-60)
  = -20

#### 例题

**例 4**：计算 8 + (-6) + (-8)

解：原式 = [8 + (-8)] + (-6）  ← 交换律 + 结合律
         = 0 + (-6)
         = -6

**例 5**：计算 16 + (-25) + 24 + (-35)

解：原式 = (16 + 24) + [(-25) + (-35)]  ← 正数结合、负数结合
         = 40 + (-60)
         = -20

#### 易错提醒

1. **乱用运算律**：减法不能直接用交换律（除非法则已经转化）
2. **结合顺序不当**：应该把相反数结合、同号结合，不要随意分组

### 2.1.3 有理数的减法

#### 一句话目标
我会把减法转化成加法：a - b = a + (-b)。

#### 为什么减法能转化成加法

**减法是加法的逆运算**：a - b 就是求一个数，使得它与 b 相加得 a。

例：3 - (-3) 就是求"什么数 + (-3) = 3"？
- 因为 6 + (-3) = 3
- 所以 3 - (-3) = 6

同时，3 + (+3) = 6，所以：
- **3 - (-3) = 3 + (+3) = 6**

#### 减法法则

**减去一个数，等于加这个数的相反数。**

公式：**a - b = a + (-b)**

#### 方法：两步转化

1. **把减号变成加号**
2. **把减数变成它的相反数**

#### 例题

**例 6**：计算 (-3) - (-5)

解：(-3) - (-5) = (-3) + 5 = 2

**例 7**：计算 0 - 7

解：0 - 7 = 0 + (-7) = -7

**例 8**：计算 7.2 - (-4.8)

解：7.2 - (-4.8) = 7.2 + 4.8 = 12

#### 易错提醒

1. **只变符号，不变数值**：-5 的相反数是 +5，不是 5（虽然数值一样）
2. **0 的特殊性**：0 - a = -a，但 0 - (-a) = +a
3. **减号变加号**：转化后不要再写减号，统一成加法

### 2.1.4 加减混合运算

#### 一句话目标
我会把加减混合统一成加法，再灵活运用运算律。

#### 统一成加法

有了减法法则，所有加减混合运算都可以**统一写成加法**。

例：(-20) + (+3) - (-5) - (+7)
- 第一步：所有减法变加法
  = (-20) + (+3) + (+5) + (-7)

#### 省略括号的形式

为了简洁，可以**省略加号和括号**（但负号不能省）：
- (-20) + (+3) + (+5) + (-7)
  = -20 + 3 + 5 - 7

**读法**："负 20，加 3，加 5，减 7"

#### 方法：同号结合、相反数结合

-20 + 3 + 5 - 7
= (-20 - 7) + (3 + 5)  ← 负数结合、正数结合
= -27 + 8
= -19

#### 例题

**例 9**：计算 (-20) + (+3) - (-5) - (+7)

解：原式 = (-20) + (+3) + (+5) + (-7)  ← 减法变加法
         = [(-20) + (-7)] + [(+3) + (+5)]  ← 负数结合、正数结合
         = (-27) + 8
         = -19

**例 10**：计算 -2.4 + 3.5 - 4.6 + 3.5

解：原式 = (-2.4 - 4.6) + (3.5 + 3.5)  ← 负数结合、正数结合
         = -7 + 7
         = 0

#### 易错提醒

1. **第一个数的正号省略**：+3 写成 3，-3 保持不变
2. **省略括号后符号要清晰**：-20 + 3 中，-20 是负二十，不是"减二十"
3. **运算顺序**：先变加法，再结合，不要跳步

### 2.1.5 绝对值的分类讨论（五星学霸特色）

#### 一句话目标
我会根据 |a| 的意义分情况讨论。

#### 为什么需要分类讨论

|a| 表示 a 到原点的距离，永远是**非负数**（≥ 0）。

但化简 |a| 时，要看 a 本身的正负：
- 如果 a ≥ 0，则 |a| = a
- 如果 a < 0，则 |a| = -a（因为 a 本身是负的，-a 就是正的）

#### 单绝对值化简

**例 11**：若 |a| = 4，b 是 3 的相反数，求 a + b

解：|a| = 4 → a = 4 或 a = -4
    b 是 3 的相反数 → b = -3

    情况 1：a = 4，b = -3 → a + b = 4 + (-3) = 1
    情况 2：a = -4，b = -3 → a + b = (-4) + (-3) = -7

    答：a + b 的值为 1 或 -7。

#### 多绝对值讨论

**例 12**：若 |m| = 3，|n| = 8，且 m, n 异号，求 |m - n|

解：|m| = 3 → m = 3 或 m = -3
    |n| = 8 → n = 8 或 n = -8
    m, n 异号 → 一正一负

    可能组合：(3, -8) 或 (-3, 8)

    情况 1：m = 3，n = -8
    m - n = 3 - (-8) = 3 + 8 = 11
    |m - n| = |11| = 11

    情况 2：m = -3，n = 8
    m - n = (-3) - 8 = (-3) + (-8) = -11
    |m - n| = |-11| = 11

    答：|m - n| = 11。

#### 易错提醒

1. **绝对值结果一定 ≥ 0**：|a| = -2 这种情况不可能
2. **不要漏情况**：|a| = 4 时，a 有两个可能（±4）
3. **异号是条件限制**：m, n 异号就排除 (3, 8) 和 (-3, -8) 的情况

## 2.2 有理数的乘法与除法

### 2.2.1 有理数的乘法

#### 一句话目标
我会"先定符号，再算绝对值"做乘法。

#### 为什么乘法有符号法则

看规律：
- 3 × 2 = 6
- 3 × 1 = 3
- 3 × 0 = 0

继续递减：
- 3 × (-1) = -3
- 3 × (-2) = -6
- 3 × (-3) = -9

**规律**：正数乘负数，积为负。

再看负数乘负数：
- (-3) × 3 = -9
- (-3) × 2 = -6
- (-3) × 1 = -3
- (-3) × 0 = 0

继续递减：
- (-3) × (-1) = 3
- (-3) × (-2) = 6
- (-3) × (-3) = 9

**规律**：负数乘负数，积为正。

#### 乘法法则（两条）

1. **同号得正，异号得负**，且积的绝对值等于乘数绝对值的积
   - 例：(+2) × (+3) = +6
   - 例：(-2) × (-3) = +6
   - 例：(+2) × (-3) = -6
   - 例：(-2) × (+3) = -6

2. **任何数与 0 相乘，都得 0**
   - 例：5 × 0 = 0，(-5) × 0 = 0

#### 方法：两步走

**"先定符号，再算绝对值"**：
- 第一步：看两数同号还是异号 → 确定结果的 + 或 -
- 第二步：乘法计算绝对值

#### 多个数相乘的符号判断

**几个非零数相乘**：
- **负数有偶数个** → 积为正
- **负数有奇数个** → 积为负

例：(-2) × 3 × (-5) × 4
- 负数有 2 个（偶数）→ 积为正
- 积 = + (2 × 3 × 5 × 4) = +120

#### 例题

**例 13**：计算 6 × (-9)

解：① 异号 → 负号
    ② |6| × |-9| = 6 × 9 = 54
    ③ 6 × (-9) = -54

**例 14**：计算 (-6) × (-1)

解：① 同号 → 正号
    ② |-6| × |-1| = 6 × 1 = 6
    ③ (-6) × (-1) = +6 = 6

**例 15**：判断积的符号：(-2) × (-3) × 4 × (-5)

解：负数有 3 个（奇数）→ 积为负

#### 易错提醒

1. **符号判断错误**：负负得正，不是负负得负
2. **0 的特殊性**：只要有一个因数是 0，积就是 0
3. **多个数相乘**：先数负数个数定符号，再算绝对值

### 2.2.2 倒数

#### 一句话目标
我会求一个数的倒数，知道倒数和乘法的关系。

#### 定义

**乘积是 1 的两个数互为倒数。**

- 例：3 × 1/3 = 1 → 3 和 1/3 互为倒数
- 例：(-5) × (-1/5) = 1 → -5 和 -1/5 互为倒数
- 例：0.25 × 4 = 1 → 0.25 和 4 互为倒数

#### 求倒数的方法

1. **整数**：分子分母颠倒
   - 5 的倒数是 1/5
   - -7 的倒数是 -1/7

2. **分数**：分子分母颠倒
   - 2/3 的倒数是 3/2
   - -3/4 的倒数是 -4/3

3. **小数**：先化分数，再颠倒
   - 0.5 = 1/2 → 倒数是 2
   - -0.25 = -1/4 → 倒数是 -4

#### 特殊情况

- **1 的倒数是 1**：1 × 1 = 1
- **-1 的倒数是 -1**：(-1) × (-1) = 1
- **0 没有倒数**：因为 0 × 任何数 ≠ 1

#### 易错提醒

1. **负数的倒数也是负的**：-5 的倒数是 -1/5，不是 1/5
2. **倒数与相反数不同**：5 的相反数是 -5，倒数是 1/5
3. **0 没有倒数**：不能说 0 的倒数是 0

### 2.2.3 乘法运算律

#### 一句话目标
我会用交换律、结合律、分配律简化计算。

#### 三条运算律

1. **乘法交换律**：ab = ba
2. **乘法结合律**：(ab)c = a(bc)
3. **乘法对加法的分配律**：a(b + c) = ab + ac

#### 应用：凑整、拆分

**技巧 1：凑整**
- 47 × (-1/8) + 81 × (1/8) + 26 × (-0.125)
  = 47 × (-1/8) + 81 × (1/8) + 26 × (-1/8)  ← 0.125 = 1/8
  = (47 + 81 + 26) × (1/8)  ← 分配律逆用
  = 154 × (1/8) - 没凑整成功，换方法
  = (-47 - 26 + 81) × (1/8)
  = (81 - 73) × (1/8)
  = 8 × (1/8)
  = 1

**技巧 2：乘法对加法的分配律**
- (-1/12 - 1/36 + 1/6) × (-36)
  = (-1/12) × (-36) + (-1/36) × (-36) + (1/6) × (-36)  ← 分配律
  = 3 + 1 + (-6)
  = -2

#### 易错提醒

1. **分配律不要漏项**：a(b + c + d) = ab + ac + ad
2. **符号要跟着走**：-a(b - c) = -ab + ac（负号要分配进去）

### 2.2.4 有理数的除法

#### 一句话目标
我会把除法转化成乘法：a ÷ b = a × (1/b)。

#### 为什么除法能转化成乘法

**除法是乘法的逆运算**：a ÷ b 就是求一个数，使得它与 b 相乘得 a。

同时，倒数告诉我们：除以一个数等于乘这个数的倒数。

#### 除法法则（两条）

1. **除以一个不等于 0 的数，等于乘这个数的倒数**
   - 公式：a ÷ b = a × (1/b)（b ≠ 0）
   - 例：6 ÷ (-3) = 6 × (-1/3) = -2
   - 例：(-6) ÷ (-3) = (-6) × (-1/3) = 2

2. **两数相除，同号得正，异号得负，并把绝对值相除**
   - 例：(-12) ÷ (-3) = +(12 ÷ 3) = +4 = 4
   - 例：12 ÷ (-3) = -(12 ÷ 3) = -4

#### 0 的除法

- **0 除以任何不等于 0 的数都得 0**
  - 0 ÷ 5 = 0
  - 0 ÷ (-5) = 0

- **任何数除以 0 都无意义**
  - 5 ÷ 0 无意义

#### 方法：两步走

**"先定符号，再算绝对值"**：
- 第一步：看两数同号还是异号 → 确定结果的 + 或 -
- 第二步：除法计算绝对值（或乘倒数）

#### 例题

**例 16**：计算 (-12) ÷ (-2)

解：方法一：① 同号 → 正号
           ② |-12| ÷ |-2| = 12 ÷ 2 = 6
           ③ (-12) ÷ (-2) = +6 = 6

     方法二：(-12) ÷ (-2) = (-12) × (-1/2) = 6

**例 17**：计算 (-1) ÷ (-5) ÷ (-1/5)

解：原式 = (-1) × (-1/5) × (-5)  ← 全变乘法
         = [(-1) × (-5)] × (-1/5)  ← 结合律
         = 5 × (-1/5)
         = -1

#### 易错提醒

1. **0 不能做除数**：5 ÷ 0 无意义，0 ÷ 0 也无意义
2. **倒数不要写错**：-5 的倒数是 -1/5，不是 1/5
3. **除法变乘法时符号要准确**：÷ (-3) 变成 × (-1/3)

### 2.2.5 有理数的混合运算

#### 一句话目标
我会按正确顺序做混合运算：先乘方，再乘除，后加减，有括号先算括号。

#### 运算顺序（三级）

1. **高级运算**：乘方（后面讲）
2. **中级运算**：乘法、除法（从左到右）
3. **低级运算**：加法、减法（从左到右）

**有括号先算括号**：小括号 → 中括号 → 大括号

#### 例题

**例 18**：计算 3 × (-4) + (-28) ÷ 7 + (-12) ÷ (3/5)

解：原式 = 3 × (-4) + (-28) ÷ 7 + (-12) × (5/3)  ← 除法变乘法
         = -12 + (-4) + (-20)  ← 先算乘除
         = -12 - 4 - 20  ← 统一加法
         = -36  ← 从左到右

**例 19**：计算 [7/12 - (0.75 + 9/20) ÷ 3.6 + 1/7] × 2 6/11

解：原式 = [7/12 - (3/4 + 9/20) ÷ (18/5) + 1/7] × (28/11)
         = [7/12 - (15/20 + 9/20) × (5/18) + 1/7] × (28/11)
         = [7/12 - (24/20) × (5/18) + 1/7] × (28/11)
         = [7/12 - (6/5) × (5/18) + 1/7] × (28/11)
         = [7/12 - (1/3) + 1/7] × (28/11)
         = [7/12 - 4/12 + 1/7] × (28/11)
         = [3/12 + 1/7] × (28/11)
         = [1/4 + 1/7] × (28/11)
         = [(7/28 + 4/28)] × (28/11)
         = (11/28) × (28/11)
         = 1

#### 易错提醒

1. **不要跳步**：一步一步算，不要心算多个步骤
2. **符号要准确**：每一步都要带符号
3. **括号先算**：不要漏掉括号

## 2.3 有理数的乘方（教材补充）

### 2.3.1 乘方的意义

#### 一句话目标
我会写乘方形式，会计算简单的乘方。

#### 定义

**求 n 个相同因数的积的运算**，叫做**乘方**。

- aⁿ 读作"a 的 n 次方"
- a 是底数，n 是指数
- aⁿ = a × a × ... × a（n 个 a 相乘）

例：
- 2³ = 2 × 2 × 2 = 8（2 的 3 次方）
- (-3)² = (-3) × (-3) = 9（-3 的平方）
- (-2)³ = (-2) × (-2) × (-2) = -8（-2 的立方）

#### 特殊情况

- **a¹ = a**（任何数的 1 次方等于它本身）
- **a⁰ = 1**（a ≠ 0，0 的 0 次方无意义）

#### 符号判断

**正数的任何次方都是正数**
- 2³ = 8，2¹⁰ = 1024

**负数的奇数次方是负数，偶数次方是正数**
- (-2)³ = -8（负）
- (-2)² = 4（正）
- (-1)¹⁰⁰ = 1（正）

**0 的任何正整数次方都是 0**
- 0⁵ = 0

#### 易错提醒

1. **-2² 与 (-2)² 不同**：
   - -2² = -(2²) = -4（先乘方，再取负）
   - (-2)² = 4（负 2 的平方）

2. **指数是几就乘几个**：2³ 是 2 × 2 × 2，不是 2 × 3

### 2.3.2 科学记数法

#### 一句话目标
我会把大数写成 a × 10ⁿ 形式。

#### 定义

**把一个数表示成 a × 10ⁿ 的形式**（其中 1 ≤ |a| < 10，n 是整数），叫做**科学记数法**。

#### 方法

1. **确定 a**：移动小数点，使它只有一位整数
2. **确定 n**：小数点移动了几位，n 就是几（原数大，n 为正；原数小，n 为负）

#### 例题

**例 20**：把 57000000 写成科学记数法

解：小数点向左移 7 位 → a = 5.7，n = 7
    57000000 = 5.7 × 10⁷

**例 21**：把 -300000 写成科学记数法

解：小数点向左移 5 位 → a = -3，n = 5
    -300000 = -3 × 10⁵

#### 易错提醒

1. **a 的范围**：1 ≤ |a| < 10，不能是 12 × 10⁵ 或 0.5 × 10⁶
2. **n 的确定**：小数点移动几位，n 就是几（包括 0）

## 本章易错点汇总

### 1. 符号错误（重灾区）

- **异号相加定错符号**：(-3) + 5 = 2（不是 -8）
- **减法转化错符号**：3 - (-5) = 8（不是 -2）
- **负负得正记反**：(-2) × (-3) = 6（不是 -6）

### 2. 运算顺序错误

- **先算低级运算**：3 + 2 × 5 = 13（不是 25）
- **括号漏算**：- (5 - 3) = -5 + 3 = -2（不是 -2）

### 3. 绝对值化简错误

- **忘讨论正负**：|a| = a 只在 a ≥ 0 时成立
- **漏情况**：|a| = 4 时，a = ±4（不只是 4）

### 4. 乘方符号错误

- **-2² ≠ (-2)²**：
  - -2² = -4
  - (-2)² = 4
