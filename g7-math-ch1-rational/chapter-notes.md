# 第一章 有理数 · 复习讲义

> 配套：人教版 2024 秋七上数学教材第一章 +《五星学霸》七上第一章
> 目标：真正理解"为什么"，能分辨易混点，会迁移应用。不是背口诀。

## 0. 本章地图（先看全局）

```
现实生活中"相反意义的量"（零上/零下、盈利/亏损、向东/向西）
        │  需要：用数区分方向相反的两种量
        ▼
   ① 正数 和 负数（0 是分界）
        │  把小学的数扩大一圈
        ▼
   ② 有理数（能写成分数形式的数）→ 分类
        │  想把数"摆"出来比较、计算
        ▼
   ③ 数轴（三要素：原点、正方向、单位长度）
        │  数 ←→ 直线上的点，一一对应
        ├──▶ ④ 相反数（关于原点对称，只有符号不同）
        ├──▶ ⑤ 绝对值（到原点的距离，永远 ≥ 0）
        └──▶ ⑥ 大小比较（右边的数 > 左边的数）
```

**一句话主线**：负数是为了表示"相反意义的量"而发明的；数轴让我们能"看见"数，于是相反数、绝对值、大小比较都变成了几何直观。

## 1. 正数和负数

### 一句话目标
我能说出：什么样的两个量是"具有相反意义的量"，并用正负数分别表示它们。

### 为什么需要负数
小学学过的数（自然数、小数、分数）都 ≥ 0，但生活里到处是**方向相反**的量：
- 零上 3℃ vs 零下 3℃
- 盈利 50 万 vs 亏损 10 万
- 向东 80 m vs 向西 60 m

光用 "3" 没法说清是"零上"还是"零下"。于是人们在原来大于 0 的数前面加一个 "-"，发明了负数，专门表示和正数**意义相反**的那一个。

### 定义
- **正数**：大于 0 的数，如 3, 50, 7.8%。有时前面加 "+" 强调（+3 就是 3）。
- **负数**：在正数前面加 "-"（负号），如 -3, -10, -0.7%。
- **0 既不是正数也不是负数**。0 是正负数的**分界**，也常常表示一个确定的基准（0℃ 是确定的温度，海拔 0 m 是海平面），不只是"没有"。

### 方法（用正负数表示相反意义的量）
1. 先选定一个**基准**（0）和一个**正方向**。
2. 与正方向同向的量记为正，反向的记为负。

### 例题
- 一箱橘子标准质量 2.5 kg。
  - 多 65 g → **+65 g**；少 30 g → **-30 g**。
  - +50 g 表示比标准**多 50 g**；-27 g 表示比标准**少 27 g**。
- 体重：李明增 1.2 kg → +1.2 kg；张华减 0.5 kg → **-0.5 kg**；刘伟不变 → **0 kg**。

### 易错提醒
- ⚠️ "增加 -0.5 kg" 不是"没变化"，而是**减少了 0.5 kg**。负号已经把方向反过来了。
- ⚠️ "不是正数的数一定是负数"——**错**。还有 0。
- ⚠️ 基准选不同，正负也不同。"比标准多"记正，和"向东记正"是两套独立规定，别混。

### 允许偏差（应用）
产品标注 `40 mm ± 0.05 mm`：标准 40 mm，合格范围 **40-0.05 到 40+0.05**，即 39.95 mm ~ 40.05 mm。

## 2. 有理数的概念与分类

### 一句话目标
我能判断一个数是不是有理数，并说出它属于哪一类。

### 定义
- 整数 = 正整数 + 0 + 负整数（如 1, 2, 3…；0；-1, -2, -3…）
- 分数 = 正分数 + 负分数（如 1/2, 2/3, 0.1, 5.32…；-5/2, -0.5…）
  - **有限小数和无限循环小数都能化成分数**，所以也算分数（0.1=1/10，0.3̇=1/3）。
- **有理数 = 整数 + 分数**，即"能写成分数形式 p/q（q≠0）的数"。

### 分类图（必须会画）
```
                  ┌─ 正整数（1,2,3…）
        ┌─ 整数 ──┼─ 0
有理数 ──┤        └─ 负整数（-1,-2,-3…）
        │        ┌─ 正分数（1/2, 0.1…）
        └─ 分数 ──┤
                 └─ 负分数（-1/2, -0.5…）

  也可按"正/零/负"三分：
  有理数 = 正有理数（正整数+正分数）、0、负有理数（负整数+负分数）
```

### 易错提醒
- ⚠️ 0 是**整数**，但既不是正数也不是负数。
- ⚠️ 百分数（8.5%）、有限小数（4.3）、循环小数（1.2̇）**都是分数**，别漏掉。
- ⚠️ "正有理数" ≠ "正整数"。正有理数范围更大（含正分数）。

## 3. 数轴（本章最重要的工具）

### 一句话目标
我会画数轴，能把任何有理数标到数轴上，也能读出数轴上的点表示什么数。

### 为什么要有数轴
有了正负数，我们想把数"摆"在一条线上看：越往右越大，越往左越小。数轴让抽象的数变成**看得见的点**，从此"相反数""绝对值""比大小"都能用图形理解（数形结合）。

### 三要素（缺一不可）
1. **原点**：直线上表示 0 的点（基准）。
2. **正方向**：通常规定向右（或向上）为正，画箭头。
3. **单位长度**：等长取一段代表 1，原点右依次 1,2,3…，左依次 -1,-2,-3…。

### 画法与标注
- 每个有理数都对应数轴上**唯一**一个点。
- 例：画数轴并表示 3, -4, 4, 0.5, 0, -5/2, -1。
  - -5/2 = -2.5，标在 -2 和 -3 正中间。

### 易错提醒
- ⚠️ 单位长度必须**均匀**；正负两侧的 1 要一样长。
- ⚠️ 分数/小数要找准位置：-5/2 在 -2 与 -3 的中点，不是 -2 旁边。
- ⚠️ 数轴向右不一定"朝东"，但**正方向一侧的数更大**是规定。

## 4. 相反数

### 一句话目标
我会求任意一个数的相反数，并理解"只有符号不同"。

### 定义
- **只有符号不同的两个数**互为相反数。如 3 和 -3，1/2 和 -1/2。
- **0 的相反数是 0**（自己和自己）。
- 数轴上看：互为相反数的两点**关于原点对称**，到原点距离相等。

### 性质（重点理解）
- 在任意一个数前面添 "-"，就得到它的相反数：
  `-(+5) = -5`，`-(-5) = +5`，`-0 = 0`。
- **a 和 -a 永远互为相反数**，不管 a 是正是负。
  - `-a 不一定是负数！` 若 a = -3，则 -a = -(-3) = 3，是正数。

### 方法（多重符号化简）
- 数 "-" 号的个数：**偶数个 "-" 结果为正，奇数个为负**（连同外面的符号一起数）。
- `-(-7) = 7`；`-(+0.5) = -0.5`；`-(-(-1))` 三个负号 → -1。

### 易错提醒
- ⚠️ "6 是相反数"——错。相反数是**两个数之间**的关系，要成对说（6 和 -6 互为相反数）。
- ⚠️ "正数和负数互为相反数"——错。必须是**只有符号不同**，如 +2 和 -3 都是正负但不是相反数（数值不同）。
- ⚠️ `-a` 不一定是负数（见上）。

## 5. 绝对值

### 一句话目标
我会求绝对值，知道它表示"到原点的距离"，且永远 ≥ 0。

### 定义（几何意义，最重要）
数轴上表示数 a 的点**到原点的距离**，叫 a 的绝对值，记作 |a|。
- |10| = 10，|-10| = 10（两者到原点都距离 10）。
- |0| = 0。

### 性质（代数意义）
- 正数的绝对值是它本身；
- 负数的绝对值是它的相反数；
- 0 的绝对值是 0。

用字母：
```
若 a > 0，|a| = a
若 a = 0，|a| = 0
若 a < 0，|a| = -a   ← 注意：a 是负数时，-a 是正数
```

### 为什么 |a| 永远 ≥ 0
因为"距离"不可能为负。这是绝对值最本质的性质：**非负性**。
- 若几个绝对值相加 = 0，则每一项都 = 0（"00 模型"）。

### 方法
- 求绝对值先判断正负，再套规则。
- |a| 最小是 0；离原点越近的点，它表示的数绝对值越小。

### 易错提醒
- ⚠️ "绝对值是它本身的数是正数"——错。**0 的绝对值也是它本身**。应是"正数和 0"。
- ⚠️ "|a| = -a 时 a 是负数"——对，但别写成 "|a| 一定是负数"。
- ⚠️ 绝对值小于 2 的整数是 ±1 和 0（共 3 个），不是只有 ±1。

## 6. 有理数的大小比较

### 一句话目标
我能比较任意两个有理数的大小，并会按顺序排列一串数。

### 法则
1. **数轴法**（最根本）：数轴上**右边的数大于左边的数**。
2. **正负法**：正数 > 0 > 负数；正数 > 负数。
3. **两个负数**：**绝对值大的反而小**。

### 为什么"两个负数绝对值大的反而小"
负数在原点左侧，绝对值越大 → 离原点越远 → 越往左 → 越小。
例如 -3 和 -7：|-3|=3 < |-7|=7，所以 **-3 > -7**（-3 在右边，更大）。

### 方法步骤
- 异号：直接用正 > 0 > 负。
- 同号（尤其两个负数）：
  1. 先分别求绝对值；
  2. 比较绝对值大小；
  3. 绝对值大的那个负数反而小。
- 含括号/符号的：**先化简**再比。如 -(-1)=1, -(+2)=-2。

### 例题（教材例 5）
- 5 和 -2：正 > 负 → 5 > -2。
- -3 和 -7：|-3|=3 < |-7|=7 → **-3 > -7**。
- -(-1) 和 -(+2)：化简为 1 和 -2 → 1 > -2。
- -(-0.5) 和 -1.5：化简为 0.5 和 1.5，都正 → 0.5 < 1.5，即 -(-0.5) < -1.5。

### 易错提醒
- ⚠️ 比较两个负数时，最容易直接比数字大小写反（以为 -7 > -3）。
- ⚠️ 含多重符号的数，**必须先化简**再比较，否则符号会看错。
- ⚠️ 排序题看清要求"从小到大"还是"从大到小"，统一用数轴检验一遍。

## 7. 本章易错点总表

| 错误想法 | 正确说法 |
|---|---|
| 0 是正数 | 0 既不是正数也不是负数，是分界 |
| 不是正数就是负数 | 还有 0 |
| -a 一定是负数 | a 为负时 -a 是正数；-a 表示 a 的相反数 |
| 相反数是一个数 | 相反数是两个数的关系，成对出现 |
| 任何正负数都互为相反数 | 必须"只有符号不同"且数值相等 |
| 绝对值是它本身→正数 | 正数和 0 的绝对值都是本身 |
| -7 > -3（数字大就大） | 两个负数：绝对值大的反而小，-3 > -7 |
| 百分数/小数不是分数 | 有限小数、循环小数都能化分数，算分数 |
