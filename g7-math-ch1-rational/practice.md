# 答案与讲解

> 每题给结果 + 关键理由。"为什么"比答案更重要。

## A 答案
- **A1.** 正数：4/3, 2.5, +14, 120。负数：-1, -3.14, -27。（0 既不是正数也不是负数）
- **A2.**
  1. ×（0 既不是正数也不是负数）
  2. ×（还有 0）
  3. ×（有 0）
  4. ×（a 为负时 -a 是正数，如 a=-3 则 -a=3）
  5. √（相反数到原点距离相等）
  6. ×（0 的绝对值也是本身；应为"正数和 0"）
  7. ×（|0|=0，应说 |a| ≥ 0，即非负数）
  8. √
- **A3.** 正有理数：{3, 13/4, 0.45, 120, 0.3̇}；负有理数：{-4/5, -7/7=-1, -2.56, -12.5}；整数：{3, 0, 120, -7/7=-1}。
  （注意 -7/7 = -1 是整数；0.3̇=1/3 是正分数）

## B 答案
- **B1.** 相反数：9/4, -6, 8, 3.5, -5/2, -10, 100, -1/3。
- **B2.** 绝对值：8, 3.9, 2/11, 100, 7.5, 0, 1/3, 1/8。
- **B3.** 1) 7  2) -0.5  3) 5/8  4) -3.8  5) -1（三个负号）  6) -3.5  7) 9。
- **B4.**
  1. 无解——|a| 表示距离，永远 ≥ 0，不可能等于 -2。（这是陷阱题）
  2. m = -10（负数且绝对值 10）。
  3. 7 的相反数 -7；-1/4 是 1/4 的相反数；相反数是本身的数是 0。

## C 答案
- **C1.** 数轴上从左到右：-4, -5/2(=-2.5), -1, 0, 0.5, 3, 4。（画图时 -5/2 在 -2 与 -3 中点）
- **C2.** 1) >（正>负） 2) >（|-3|=3<|-7|=7，绝对值小反而大） 3) <（-(-2.25)=2.25，-2.5<2.25） 4) >（|-3/5|=0.6<|-3/4|=0.75） 5) <（-(+8)=-8，-(-9)=9，-8<9） 6) >（-(-0.3)=0.3，0.3 > -1/3≈-0.333）
- **C3.** -10 < -3 < 0 < 2 < 5 < 8
- **C4.** -3/2 < -1/2 < -1/4 < -0.3 < 0 < +2.3
  （-3/2=-1.5, -1/2=-0.5, -1/4=-0.25, -0.3；注意 -0.3 < -0.25 因为 |-0.3|>|-0.25|）
- **C5.** B = 1 或 -7。（向右移 4：-3+4=1；向左移 4：-3-4=-7）
- **C6.** 整数点：-2,-1,0,1,2,3,4 共 7 个；负整数 -2,-1 共 2 个。
- **C7.** a<0, b>0, |a|<|b| → -b < a < 0 < -a < b。即从小到大：**-b < a < -a < b**。

## D 答案
- **D1.** 1) +65 g、-30 g。 2) +50 g：比标准多 50 g；-27 g：比标准少 27 g。
- **D2.** 最大 40+0.05 = 40.05 mm；最小 40-0.05 = 39.95 mm。范围 39.95 ~ 40.05 mm。
- **D3.** 7 - 4 - 4 = **-1℃**。
- **D4.** 31.5 > 27.8 > -6.8 > -10.7。
- **D5.** 100×7 + (4-3-5+14-8+21-6) = 700 + 17 = **717 斤**。
- **D6.**
  1. 向西为负：A = -2，B = -5，C = +4（-5 + 9 = 4）。
  2. C - A = 4 - (-2) = **6 km**。
  3. 总路程 = 2+3+9+(回邮局 4) = 18 km；耗油 = 18 × 0.03 = **0.54 L**。

## E 答案
- **E1.** 选 **C：-3 或 11**。
  |a|=4 → a=±4；|b|=7 → b=±7；a-b<0 → a<b。
  - a=4：需 4<b，b=7 ✓ → a+b=11。
  - a=-4：需 -4<b，b=7 ✓ → a+b=3；b=-7 时 -4<-7 ✗。
  - 重算：a=-4,b=7→3；a=4,b=7→11。但 3 不在选项 C 的"-3 或 11"。仔细查：a-b<0 即 a<b。
    组合：(a,b)=(4,7)→和11，a<b✓；(4,-7)→-3，4<-7✗；(-4,7)→3，-4<7✓；(-4,-7)→-11，-4<-7✗。
    满足的：11 和 3。选项中含"3"的是 D（11或3）。**正确答案 D**。
  > 教训：分类讨论要穷举四种组合再筛选，别漏。
- **E2.** 选 **C：-7 或 1**。b = -3（3 的相反数）；|a|=4 → a=±4。a+b = 4+(-3)=1 或 -4+(-3)=-7。
- **E3.** m、n 异号：(|m-n| 取最大差) m=3,n=-8 → |3-(-8)|=11；m=-3,n=8 → |-3-8|=11。答案 **11**。
- **E4.** |x|=2 → x=±2（不一定是 2）；|x|=0 → x=0；x=-x → x=0（只有 0 等于自己的相反数）。
- **E5.** -1 与 0 之间有负数（如 -0.5）；0 与 1 之间无负数（都是正）；-3 与 -1 之间负整数：-2；小于 -100 且大于 -103 的数：-101, -102, -101.5（3 个示例）。
- **E6.** 设 P=p, Q=p+9, C=-1。‖PCQ‖ = ||P-C| - |Q-C|| = ||p+1| - |p+10|| = 6。
  因为 PQ=9，|p+1| 与 |p+10| 之差的绝对值最大为 9（两点分别在 C 两侧时）。
  讨论 p 的区间（只有当 p 介于 -10 和 -1 之间，即 C 落在 P、Q 之间时才有解）：
  此时 |p+1| = -p-1，|p+10| = p+10，于是 |-2p-11| = 6 → -2p-11 = ±6 → **p = -8.5 或 p = -2.5**。
  （注：P 表示的数可以是小数；若限定整数则本题无整数解。）
